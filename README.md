## 💱 Currency Converter

A fast and user-friendly currency converter application that provides real-time exchange rates. This project allows users to instantly convert amounts between different currencies using live market data. Built with a focus on accuracy and clean UI design.

## 🚀 What This Converter Can Do

- 💲 Convert custom amounts instantly
- 🔁 Swap currencies with one click
- 📱 Fully responsive design (mobile & desktop friendly)
- ⚡ Fast and lightweight performance

## 🛠️ Technologies Used

- HTML5
- CSS3
- React
